# GSAP Site

پروژه‌ی قابل ویرایش Fort Energy با Vite، React، Tailwind CSS، GSAP و موتور WebGL اصلی.

## اجرا

```bash
npm install
npm run dev
```

تمام متن‌ها، نام‌ها، لینک‌ها و عنوان‌ها در `src/data/siteData.js` قرار دارند. کامپوننت‌های صفحه در `src/components` هستند. فایل‌های GLB و WebGL داخل `public` نگهداری می‌شوند و برای تغییر محتوا نیازی به ویرایش آن‌ها نیست.
