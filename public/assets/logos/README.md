# Logo assets

This folder is reserved for approved runtime logo files served from `/assets/logos/`.

The current production hero and footer marks are not controlled solely by files in this folder: the protected legacy HTML contains inline SVG, and `LegacySite` applies a footer safeguard. Do not assume adding a file here changes production.

Before adding or replacing a logo:

1. record provenance and ownership;
2. use a brand-neutral lowercase kebab-case filename;
3. register the asset in the approved asset registry when M3 is complete;
4. update only the renderer explicitly included in the approved milestone;
5. verify desktop, tablet, mobile, loading, and fail-closed branding against M0.
